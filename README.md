# 📱 Smart Coaching Tracker App

## 📌 Project Idea

The Smart Coaching Tracker App is a mobile application built using Flutter and Firebase that helps rural coaching centers efficiently track student attendance and academic progress. It provides real-time updates, analytics, and improves communication between teachers and students.

---

## 🚀 Features (Planned)

* Student Attendance Tracking
* Academic Performance Monitoring
* Teacher Dashboard
* Notifications System
* Google Maps Integration for location tracking

---

## 🛠️ Tech Stack

* Flutter (Frontend)
* Firebase Auth
* Firestore Database
* Cloud Functions
* Google Maps API

- POST → Write to DB using save()
- PUT → Update using findByIdAndUpdate()
